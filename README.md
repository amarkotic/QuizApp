# QuizApp - built in 2021

 An educational iOS app that personalizes learning through quizzes on various topics.

 ## Technologies used
• UI - UIKit + Snapkit

• Dependencies - Cocoapods

• Local database - Core data

• Architecture - MVP (Model-View-Presenter)

• Restful API
